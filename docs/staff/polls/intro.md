Pawtograder's Polls feature provides a quick and easy way for instructors to conduct live in-class polling. Polls are designed for real-time engagement during lectures, allowing instructors to gauge student understanding, gather quick feedback, or facilitate interactive discussions.

## Key Features

- **Quick Creation**: Create polls in seconds with a simple interface
- **Real-Time Results**: See student responses as they come in
- **Multiple Poll Types**: Support for various question formats
- **Live Visualizations**: Display results to the class with built-in visualizations
- **Automatic Closing**: Polls automatically close after one hour to ensure timely responses

## Use Cases

Polls are ideal for:
- **Comprehension Checks**: Quickly assess if students understand a concept before moving on
- **Discussion Starters**: Gather opinions to spark class discussions
- **Attendance**: Quick polls at the start of class
- **Feedback**: Get instant feedback on pace or clarity of instruction

## Getting Started

To use Polls, navigate to your course and select "Polls" from the course navigation. From there, you can create new polls and view results from previous polls.

:::note
Polls are scoped to your class - you can only view and manage polls for courses you are an instructor for.
:::
